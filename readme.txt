== Description ==

Create order forms and quote forms with modern layout and multiple options including but not limited to dropdown images, text inputs, images, checkboxes, hyperlinks and much more. Best for online sellers to take bulk orders from their clients. Get notified via email and print / export orders to pdf.

== Features ==

1. Order Forms with modern layout in WordPress
2. Add thumbnails / images of the product
3. Different input options including images, text fields, hyperlinks, non-editable fields, dropdown options, checkboxes and much more.
4. Option to print orders or exporting order to pdf
5. Customize Your Form Notifications
6. Option to set custom quantatites based on nature of the product
7. Option to set custom sizes based on nature of the product

== Installation ==

Installation of the plugin is really simple.

1. Install using one of these options:
    * Install directly from WordPress Admin panel: go to Plugins -> Add New -> Search for "Stylish Order From Builder", and click the Install button.
    * Download the ZIP manually from WordPress' plugins repository, and upload it through WordPress Admin panel: go to Plugins -> Add New -> Upload Plugin, browse to the downloaded Zip and upload it.
    * Download the ZIP, extract it and manually upload the extracted folder through FTP to the `/wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!
